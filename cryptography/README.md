# Cryptography CLI Tool

A Python command-line application that demonstrates classical encryption methods such as the **Caesar cipher**.

## Features
- Interactive menu
- Caesar cipher encryption
- Save encrypted results (text, key, cipher) to files
- Automatic key file generation
- Clean CLI interface (colors + ASCII art)

## Technologies
- Python 3
- pyfiglet
- colorama

## Usage
Run the main script and follow the on-screen menu:
```bash
python main.py
```
This is an educational project built to learn Python fundamentals, modular code, and basic cryptography.
